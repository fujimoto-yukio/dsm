# DSM Workbench
DSM Workbenchは、DSM (Design Structure Matrix) の編集と分析を行うためのツールで、二値型（Binary）と数値型（Numeric）の2種類のDSMを扱います。
## はじめに
DSM Workbenchと名付けて、DSM(Design Structure Matrix)の編集・分析ツールを公開することにしました。
ここでは、このツールの生い立ちから、開発、公開に至った経緯を簡単に説明させてください。

開発者である私は1961年生まれ（これを書いている現在63歳）で、元々は機械エンジニアでしたが、SI業界に転職して以来、主に設計開発領域のSIを担当させていただきました。
60歳定年を2年後に控えた2019年1月、（それまでお客さまだった）自動車メーカーの情報システムに出向し、社内SEとして自動車の設計支援システムを中心に、（楽しく）仕事をさせていただきました。その活動の中のひとつに、DSMの設計活用というテーマもありました。
出向から1年と少し経過した2020年にコロナ禍が発生し、この年のゴールデンウィークは、緊急事態宣言で自宅引きこもりとなりました。あまりに暇でやることがない連休を持て余し、思いついたのが「DSMを画面で操作できるアプリを作ってみよう！」でした。

SI業界に転職した当初はJavaのプログラミングもバリバリやっていましたが、月日が経つにつれて管理の仕事が中心で、ちょこっとしたツールぐらいしか作らなくなっていたところに、随分と進化したJava（ラムダ式は、サンプルコードを見てもまるで意味不明でした）と（GUIアプリなんか作ったこともないのに）初めてのJavaFXを使い、DSMアプリのプロトタイプを作りました。出来上がったアプリは、CSVからDSMを取り込み、要素を上下（左右）に移動、並び替えるだけの簡単なものでしたが、画面上でDSMを直接操作し整理することができる、画期的なアプリに思えました。

その後、2022年9月末に出向から復帰するまでの間、仕事の暇を見つけて、画面の見た目をよくし、操作性を上げ、可到達マトリクスやグラフデータベースと連携したコミュニティ分析等、いくつかの機能を追加しました。設計現場でも使ってもらいましたが、ベースが1週間で作ったプロトタイプでもあり、業務に本格適用することのできるものにはなりませんでした。

出向から自社復帰してから1年後の2023年9月末、仕事のきりを付けて63歳半で退職しました。
ゴールデンウィークよりもはるかに長い退職後の時間を何に使うか考えた時、やりっぱなしになっているDSMツールをきちんと作り直すことに決めました。JDK1.4か1.5で止まっているJavaの知識に、Java21までのJavaの変化を学び、JavaFXによるGUIアプリ開発も学んで、新しいアプリケーションとしてDSM Workbenchを開発してきました。設計書は多くはありませんが、設計もしました。

どうしても基盤機能の比重が大きくなる中、プログラムを書くだけでは楽しみが少ないので、個人的な興味で機能を作り込んだところも多々あります。特に、グラフデータベースを使う機能とグラフ描画の機能に関しては、グラフ理論を理解するという基本を疎かにして、「何か使えそう」というか、「出来上がったものを見るのが楽しい」ということを判定基準に機能を作り込みました。

以上のような経緯で出来上がったDSM Workbenchの機能には、意味のあるもの、ないもの玉石混合になっていますので、それを理解した上で使ってください。作者としては、玉が少しでも混ざっていて、何かのお役に立てることを期待しています。
また、ひとりで設計、開発、テストを行ってきましたので、バグ、不具合もまだまだ残っているはずです。ご連絡いただければ、可能な限り修正しますので、これも覚悟の上使っていただければと思います。
あくまで「As Is」ですが、これらをご理解の上で利用していただければ幸いです。
## 機能説明
### DSM Workbench
  - アプリケーションを起動すると、最初に表示される画面です。
  - DSMの種類を指定して、新規のDSMを編集するための`DSM Editor`を起動することができます。
  - ファイルを指定して`DSM Editor`を起動することができます。
  - ファイルを指定して`Graph Visualizer`を起動することができます。
    <img width="712" alt="スクリーンショット 2024-10-09 20 25 13" src="https://github.com/user-attachments/assets/98b0f952-4345-4b41-b90d-32d55dc5440b">
### DSM Editor
  - DSMの編集と分析を行うための画面です。
  - `二値型（Binary）`と`数値型（Numeric）`の2つのDSM形式を取り扱います。
  - 操作モードには、`セル編集モード`と`マトリックス編集・解析モード`の2つのモードがあります。
  - `セル編集モード`では、DSM要素の定義、属性編集とDSM要素間のフィード値を編集することができます。
  - `マトリックス編集・解析モード`では、DSM行列に対する移動、並び替え、部分抽出等の操作と、DSMの順序付けやコミュニティ分析等の解析を行います。
    <img width="1112" alt="スクリーンショット 2024-10-09 20 24 27" src="https://github.com/user-attachments/assets/ce3c534f-70be-48a4-ba24-576a4ed5b702">    
### Graph Visualizer
  - `DSM Editor`で作成したDSMをネットワーク・グラフに描画します。
  - `JUNC Visualizer`と`SmartGraph Visualizer`の2種類のグラフ描画形式を選択することができます。
    <img width="922" alt="スクリーンショット 2024-10-09 20 27 21" src="https://github.com/user-attachments/assets/c5a50f20-0f67-4db4-a477-36470e7e1035">
### Preference Dialog
  - `DSM Workbench`の各種設定値を管理します。
  - 設定値はユーザ毎に保存されます。
    <img width="1117" alt="スクリーンショット 2024-10-09 20 27 36" src="https://github.com/user-attachments/assets/0140a362-64e3-4654-907c-ec7cfe9ddcb1">
### Information DIalog
  - `DSM Workbench`で取り扱う各種情報を詳細表示します。
  - 画面の表示内容をファイルに保存することもできます。
